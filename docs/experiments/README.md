# 实验日志（Experiments）

系统化、可复现地记录每次策略/信号/prompt 的改动与结果，避免"改了很多但说不清哪个有用"。

## 纪律

- **单变量**：一次只改一个变量。
- **对照基准**：始终与基准或上一版对比。
- **可复现**：记录数据版本、代码版本、模型/prompt 版本、随机种子、参数。
- **诚实**：无论结果好坏都记录；负面结果同样有价值。
- **防过拟合**：结论必须以样本外表现为准，标注是否触及最终验证集。

## 命名

`YYYYMMDD-简短名.md`，用 [experiment-template.md](experiment-template.md) 起草。
